# LinuxCalculator

A foundational CLI application used to verify C# build processes and compilation on a Linux environment.

## Purpose
This was my initial proof-of-work project to ensure the .NET SDK was correctly configured and to understand the workflow of compiling and executing C# code on a Linux server.

## Features
*   **Basic Arithmetic**: Implements fundamental operations to test application logic.
*   **CLI Interface**: Demonstrates handling user input/output in a headless environment.
