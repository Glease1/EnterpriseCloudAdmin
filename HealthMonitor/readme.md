# HealthMonitor

A foundational C# project developed to explore system monitoring and file-based logging capabilities on Linux.

## Purpose
This tool was built to practice interacting with the filesystem and managing log output. It serves as a proof-of-work for implementing automated monitoring logic in a C# environment.

## Key Features
*   **System Logging**: Captures health data and writes it to `system_health.log`.
*   **File I/O**: Demonstrates reading and writing operations within a Linux directory structure.
